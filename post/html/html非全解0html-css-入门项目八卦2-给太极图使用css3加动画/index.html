为什么没有红线没和页面左面重合？

  • 因为body有默认内边距样式，要干掉：

    1
2
3
4
5

    * {
box-sizing: border-box;
padding:0;
margin:0;
}

vh

红框高度为什么不和页面一样高？height:100%没用？

  • 在body下写height:100%没有用，为什么？google

  • 要写成height:100vh

    1
2
3
4

    #八卦-wrapper{
border:1px solid red;
height:100vh;
}

vh: viewport height 用户可视范围 窗口左上到右下

  • 100（即100%）用来表示和可视范围一样高（宽度不用100%，自适应即可）
  • 不能写成100%vh，这样只认100%，vh失效
