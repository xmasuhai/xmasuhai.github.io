如何使用Switch组件

  • <Switch value="true"/>，value为字符串"true"
  • <Switch value="false"/>，value为字符串"false"
  • <Switch :value="true"/>，value为布尔值"true"

  • 当value为字符串"true"或布尔值true时，显示为开
  • 其他情况一律显示为关

<script setup lang="ts">
import {ref, reactive} from 'vue';
const count = ref(100); // 简单类型 数据引用
const object = reactive({n: 999}); // 复杂类型 // 使用数据代理：object 是 {n: 999} 的代理
setTimeout(() => {
  count.value += 100;
  object.n += 1
}, 3000)
</script>

<script lang="ts">
export default {
};
</script>
  • ref()加简单类型，reactive()加复杂类型
  • ref() 创建基础类型的响应式数据对象 const data = ref(1)
    • 模板默认隐式调用 value 显示数据 {{ data }}
    • 方法中修改需要修改 value 的值才能修改 data.value = 2
  • reactive() 创建引用类型的响应式数据对象
    • reactive() 将每一层的数都解析成proxy对象
    • 响应式默认都是递归的，改变某一层的值都会递归的调用一遍，重新渲染 dom
  • 使用ref的值，需要在变量后加.value，reactive则直接使用
