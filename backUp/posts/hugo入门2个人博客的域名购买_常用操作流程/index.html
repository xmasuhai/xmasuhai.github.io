【Hugo入门2】个人博客的域名购买_常用操作流程

December 19, 2019

买域名

  • namesilo（方便）
  • 阿里云（中文）
  • 备用 华为/腾讯/…

namesilo

  1. 选yourName.xyz，0.99$ a year 7元玩一年，续费会贵一点，网上有优惠折扣
  2. 选项：none/No/NoPrivacy/1
  3. continue支付宝购买/邮箱验证/
  4. Domain Manager
  • 点击：Manage My Domain
  • 点击地球图标：Manage DNS for this domain

阿里云

域名与网站>域名注册 第二年续费会坑你钱

配置DNS

  • 删掉现有的无关配置
  • 配置四条A记录到185.199.108.153等IP
  • 确定配置成功（ win：nsloop域名 / mac：dig + noall + answer 域名）

  • A记录可能要等几天

    nslookup yourName.xxx

    dig noall answer yourName.xxx

配置GitHub Pages

  • 添加域名 进入你博客的git仓库>Settings 下拉到GitHub Pages>Custom domain 输入买好的域名，我的xmasuhai.xyz

  • 不开启HTTPS比较方便测试 取消勾选Enforce HTTPS 否则你与此相关网站都必须是HTTPS，申请免费的HTTPS证书

设置顶级域名：apex domain 拷代码时关掉翻译 看到四个IP > To create an A record, point your apex domain to the IP addresses for GitHub Pages.

185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

@ IP 确定

  • 能成功访问你的域名，就说明完成了一个前端第一次将自己的域名和内容上传到万维网上，可以上一炷香给李爵爷了
  • 突然有一天，访问不了自己域名， 把git远程仓库的CNAME文件给删了， 或者在Settings里删掉Custom domain，或者续费域名
