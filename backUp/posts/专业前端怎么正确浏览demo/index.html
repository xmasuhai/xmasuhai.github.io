专业前端怎么正确浏览Demo

December 22, 2019
  • 没有人浏览网页是双击打开xxx.html文件，这不是用户通常的操作，特别是专业前端绝对不允许这样做，可能会遇到奇奇怪怪的问题
  • 必须通过输入网址来打开预览页面

两个工具模拟打开网页

1.用http-server

用yarn安装：

yarn global add http-server
...
hs --version

怎么用？

cd ~/xxxx # 进入目录下
http-server . -c-1
  • -c代表缓存，-1代表不需要，让http-server在无缓存的模式下运行

省略写法（省略”http-server”和”.“）：

cd ~/yourDemo
hs -c-1

显示：

Starting up http-server, serving ./
Available on:
  http://127.0.0.1:8080
  http://192.168.0.105:8080
Hit CTRL-C to stop the server

按住Ctrl/CMD点击运行任何一个访问端口，加路径/xxx.html

提问bug代码.html，不要双击打开，要在终端里通过http-server模拟，用网址来访问一个网页后，再提问

为什么我的hs只能显示两个端口？

2.用parcel

  • 安装parcel：

    yarn global add parcel

在目录下，直接用

parcel yourDemo.html

访问开启的地址，像用户一样，开始浏览，工具能用就行
