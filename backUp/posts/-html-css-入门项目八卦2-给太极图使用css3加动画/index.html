HTML & CSS 入门项目八卦2-给太极图使用CSS3加动画

December 18, 2019

让八卦转：

动画属性：

#八卦 {
    position: relative;
    width: 400px;
    height: 400px;
    border-radius: 50%;
    overflow: hidden;
    animation: x 3s linear infinite ;
    }

animation x 1s;，声明动画名称x（标识符identifier）

  • 遇坑，勿写中文：，看清:没用
  • 输入法隔离，IDE专用英语输入法，开一个便签，需要中文，粘贴进来

写关键帧：

@keyframes x{
    from{
        transform:rotate(0deg);
    }
    to{
        transform:rotate(306deg);
    }
}

  • 关键位置from和to

  • 顺序不重要：animation: x 3s linear infinite ;

调阴影：

搜索：box-shadow generator预览阴影样式，再誊写回来

遇事不知搜*** generator，好处是能帮你生成代码

  • 工具：cssmatic:
  • Blur Radius 不超过5px
  • 调代码，然后copy
#八卦 {
    position: relative;
    width: 400px;
    height: 400px;
    border-radius: 50%;
    overflow: hidden;
    animation: x 3s linear infinite ;

    }

！命名规则：包裹作用的一般叫：**-wrapper

CSS Border法调试：
#八卦-wrapper{
    border:1px solid red;
}

! 先确定位置，在写CSS样式:

#八卦-wrapper{
    border:1px solid red;
}

为什么没有红线没和页面左面重合？ - 因为body有默认内边距样式，要干掉：

* {
    box-sizing: border-box;
    padding:0;
    margin:0;
}

vh

红框高度为什么不和页面一样高？height:100%没用？ - 在body下写height:100%没有用，为什么？google

  • 要写成height:100vh

    #八卦-wrapper{
border:1px solid red;
height:100vh;
}

vh: viewport height 用户可视范围 窗口左上到右下 - 100（即100%）用来表示和可视范围一样高（宽度不用100%，自适应即可） - 不能写成100%vh，这样只认100%，vh失效

flex部个局

内容居中：

#八卦-wrapper{
    border:1px solid red;
    height:100vh;    
    /* flex三行式 */
    display: flex;
    justify-content: center;
    align-items: center;
}

  • 字怎么在右边 > 字会往上围绕 > - 用div包起来，和#八卦div形成上下结构

    <div id="八卦-wrapper">
<div id="八卦">
    <div></div>

    <div></div>

</div>
<div>
    道可道，非常道
</div>
</div>
  • 但此时还受display:flex影响，默认把里面的元素横向一字排开：
#八卦-wrapper{
    border:1px solid red;
    height:100vh;    
    /* flex三行式 居中*/
    display: flex;
    justify-content: center;
    align-items: center;
    flex-direction: row;
}

不对，遇错，试试另外的属性选项，二选一/三选一，马上换

#八卦-wrapper{

    height:100vh;

    display: flex;
    justify-content: center;
    align-items: center
    flex-direction: column;
